HTML Structure

This template uses the 1170 grid system (12 columns) which will help streamline any customization you wish to make. If you're unfamiliar with the 1170 grid system, be sure to read about it here!

The main structure of this site is wrapped in a 'container' div. Inside of that div you will find each major section of the site given an appropritae grid size all equalling 12 columns in length. For example, the '#abut_event' element has a class of 'col-sm-8' and the '#event_images' element has a class of 'col-sm-4'. The total columns equals 12 columns.

                <section class="section-padding">
                    <div class="container">	
                        <div class="row">
                            <div id="abut_event" class="col-sm-8"></div>
                            <div id="event_images" class="col-sm-4"></div>
                        </div>
                    </div>
                </section>

Note: If you want to add padding in section use class="section-padding" 140px padding top & 140px padding bottom added in section. You don't need padding remove class="section-padding".

Countdown

You can change the event countdown time with change this code in (assets/js/interface.js):

                	/*------------------------------------------------------------------
                        Countdown
                        -------------------------------------------------------------------*/
                        
                    $(function(e) {
                            var endDate = "march 22, 2017";
                            $('.countdown.styled').countdown({
                              date: endDate,
                              render: function(data) {
                                $(this.el).html("
" + this.leadingZeros(data.days, 2) + " Days
" + this.leadingZeros(data.hours, 2) + " Hours
" + this.leadingZeros(data.min, 2) + " Minutes
" + this.leadingZeros(data.sec, 2) + " Seconds
");
                              }
                            });
                    
                          });

Note: Change date time automatically set.

How to Change Mail Chimp Api Key

Below is some php code to change the Mail Chimp API KEY & List Unique ID settings (inc/store-address.php):

  1. Open the store-address.php file.
  2. Find below highlighted line number 12 and 16 code in store-address.php file and change "API KEY" and "List Unique ID": 
                        	function storeAddress(){
	
                        // Validation
                        if(!$_GET['email']){ return "No email address provided"; } 
                    
                        if(!preg_match("/^[_a-z0-9-]+(\.[_a-z0-9-]+)*@[a-z0-9-]+(\.[a-z0-9-]+)*$/i", $_GET['email'])) {
                            return "Email address is invalid"; 
                        }
                    
                        require_once('MCAPI.class.php');
                        // grab an API Key from http://admin.mailchimp.com/account/api/
                        $api = new MCAPI('your_apikey'); // "API KEY"
                        
                        // grab your List's Unique Id by going to http://admin.mailchimp.com/lists/
                        // Click the "settings" link for the list - the Unique Id is at the bottom of that page. 
                        $list_id = "my_list_unique_id"; // "List Unique ID"
                    
                        if($api->listSubscribe($list_id, $_GET['email'], '') === true) {
                            // It worked!	
                            return 'Success! Check your email to confirm sign up.';
                        }else{
                            // An error ocurred, return error message	
                            return 'Error: ' . $api->errorMessage;
                        }
                        
                    }

CSS Structure

I'm using one main CSS file (assets/css/style.css), in this template which you are free to modify depending on the customization you require. Below are 4 additional CSS files:

  • bootstrap.min.css - This is a main stylesheet that required for basic template styling like grid, form, input and web elements.
  • font-awesome.min.css - CSS stylesheets for font icon packs.
  • owl.carousel.css - This stylesheet used for styling the Banner slideshow.
  • owl.transitions.css - This stylesheet used for styling the Banner transitions.
